## 字段说明

| 字段 | 含义 |
| --- | --- |
| 费用 | 打出消耗的能量 |
| 生命 / 施法数 | 仆从为生命值；普通法术为**最多可打出次数**；仪式法术为占位**可执行次数** |
| 攻击 | 仆从攻击力；法术通常为空 |
| 效果 | 卡牌正文 |
| 强化 1 / 2 | 强化路线上的数值或效果变化 |
| TAG | 标签（大型、地狱等） |
| 稀有度 | 普通 / 稀有 / 珍贵 / 传奇 |
| 专用 | 限定英雄（如地狱术士） |
| 父记录 | Token 的来源卡 |
